# TinyLink Frontend (React + Vite)

TinyLink is a simple and fast URL Shortener frontend built using **React**, **Vite**, **Context API**, and **CSS Modules**.  
This frontend connects to your backend API to create short links, manage links, authenticate users, and display analytics.
